Introduction

To access the data, you can either use the manager or directly the Eloquent model.

If you wish to extend any class, remember to update the configuration. You can publish the configuration through the command php artisan vendor:publish

Let's first check the list of all attributes

Manager

Base class Railken\Amethyst\Managers\WorkManager

The manager is the main class to access and manipulate your model.

Why use the manager instead of the model? Because the manager handle all the boring stuff like validation and authorization for you. Remember that the manager return always a Result.

You can extends the class like the following example in app/Managers/WorkManager

namespace App\Managers;

use Railken\Amethyst\Managers\WorkManager as Manager;

class WorkManager extends Manager {
    // ...
}

Remember to update the configuration with new class.

Create a new entity

First you have to define a new instance of the Manager

use Railken\Amethyst\Managers\WorkManager;

$manager = new WorkManager();

Now you can use the method create to create a new a new entity

$params = [
    "name" => "Hester Jakubowski",
    "payload" => "dummy: dummy\n",
    "description" => "Ea in ratione accusantium soluta omnis. Aut corrupti aliquid magni. Perferendis est perspiciatis eius dolorum quis quia et.",
    "data_builder" => [
        "name" => "Fleta Abbott",
        "description" => "Et eos labore ut repellat. Sunt omnis odit doloremque.",
        "class_name" => "Railken\\Amethyst\\DataBuilders\\CommonDataBuilder",
        "class_arguments" => "- Railken\\Amethyst\\Managers\\DataBuilderManager\n",
        "filter" => "id eq 1",
        "input" => "date:\n    type: date\n    validation: 'date_format:Y-m-d'\n",
        "mock_data" => "date: '2018-01-01'\n",
        "include" => "bar",
        "data_builder" => [
            "class_name" => "Railken\\Amethyst\\DataBuilders\\DummyDataBuilder"
        ]
    ],
    "data" => "id: '{{ record.id }}'\n"
];

$result = $manager->create($params);

Check the result of the operation

if ($result->ok()) {
    // All ok
} else {
    // Something goes wrong
}

Retrieve the entity from the result

$entity = $result->getResource();

Result

Once you've got the result you should always check if an error has occurred, if not, retrieve the resource.

use Railken\Amethyst\Managers\WorkManager;

$manager = new WorkManager();

$result = $manager->create([
    "name" => "Hester Jakubowski",
    "payload" => "dummy: dummy\n",
    "description" => "Ea in ratione accusantium soluta omnis. Aut corrupti aliquid magni. Perferendis est perspiciatis eius dolorum quis quia et.",
    "data_builder" => [
        "name" => "Fleta Abbott",
        "description" => "Et eos labore ut repellat. Sunt omnis odit doloremque.",
        "class_name" => "Railken\\Amethyst\\DataBuilders\\CommonDataBuilder",
        "class_arguments" => "- Railken\\Amethyst\\Managers\\DataBuilderManager\n",
        "filter" => "id eq 1",
        "input" => "date:\n    type: date\n    validation: 'date_format:Y-m-d'\n",
        "mock_data" => "date: '2018-01-01'\n",
        "include" => "bar",
        "data_builder" => [
            "class_name" => "Railken\\Amethyst\\DataBuilders\\DummyDataBuilder"
        ]
    ],
    "data" => "id: '{{ record.id }}'\n"
]);

if ($result->ok()) {

    $resource = $result->getResource();

} else {

    // Loop through all errors
    $result->getErrors()->map(function($error) {
        return $error->toArray();
    }))

    // Retrieve an array of all errors
    $result->getSimpleErrors();

    /* The result is something like this:

        [0] => Array
            (
                [code] => FIELD_NOT_DEFINED
                [label] => field
                [message] => The field is required
                [value] =>
            )
    */

}

Repository

Base class Railken\Amethyst\Repositories\WorkRepository

The repository is the class used to perform queries.

use Railken\Amethyst\Managers\WorkManager;

$manager = new WorkManager();

$repository = $manager->getRepository();

Retrieving an entity

$repository->findOneBy(['id' => 1]);
$repository->findOneById(1);

Retrieving all entities

$repository->findAll();

Performing a query using \Illuminate\DataBase\Eloquent\Builder

$repository->newQuery()->where('id', 1)->first();

You can extends the class like the following example in app/Repositories/WorkRepository

namespace App\Repositories;

use Railken\Amethyst\Repositories\WorkRepository as Repository;

class WorkRepository extends Repository {
    // ...
}

Remember to update the configuration with new class.

Serializer

Base class Railken\Amethyst\Serializers\WorkSerializer

The serializer is used to serialize an entity, you can retrieve it from the data.

use Railken\Amethyst\Managers\WorkManager;

$serializer = $manager->getSerializer();

And use it so serialize an entity Retrieving an entity

$entity = $repository->findOneById(1);
$serializer->serialize($entity)->toArray(); // Returns an array
